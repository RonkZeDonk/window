# Window
A window view of your PC

## Current Features
- See what's playing on you computer and change tracks with a cross platform Flutter app (app coming soon)
- Desktop GUI client (coming soon)

## Requirements
- A Windows PC/laptop
- A phone/tablet
- A network connection
